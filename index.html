Google Cloud region recommender

Helper tool to pick a Google Cloud region, taking carbon footprint, price and latency into account. Read more about Google Cloud [sustainability efforts](https://cloud.google.com/sustainability).

Data is never uploaded. All processing is happening client side in your browser. This tool is on GitHub.

This is not an officially supported Google product.

Results

    Carbon footprint is approximated using the yearly Carbon free energy percentage as well as the average carbon intensidy of the grid, read more.
    Latency is approxiated using physical distance between selected countries and country of the region.
    Price is approximated using Google Compute Engine E2 instance vCPU price.
